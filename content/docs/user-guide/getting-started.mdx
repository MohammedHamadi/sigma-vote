# Getting Started

As an administrator, you can create a new election from the dashboard.

1. Navigate to the Admin Dashboard
2. Click "Create Election"
3. Enter the election title, description, start time, and end time
4. Add candidates
5. The system will automatically generate a new Paillier keypair for the election
6. Publish the election
